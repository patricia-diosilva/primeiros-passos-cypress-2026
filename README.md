# My Cypress Automation

This project utilizes Cypress to automate the testing of a login feature and a user update feature. Cypess is a Javascript-based end-to-end testing framework that allows developers to write tests and execute them in a real browser.


## Instalation
```bash
npm install
```

>**NOTE:**
>It is necessary to have **node** installed


## Running
```bash
#You open the Cypress UI
npx cypress open

# Run the automation by command line
npx cypress run
```
